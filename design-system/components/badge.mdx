# Badge

## Purpose

Small label for status, category, or count indicators. Built with CVA.

## Source

`src/components/ui/badge.tsx`

## Variants

| Variant | Appearance | Usage |
|---|---|---|
| `default` | Brand primary fill | Default labels |
| `secondary` | Muted background | Low-emphasis labels |
| `destructive` | Red fill | Error/warning indicators |
| `outline` | Border only | Neutral tags |

## States

- **Default**: Static display element
- Badges are not interactive by default
